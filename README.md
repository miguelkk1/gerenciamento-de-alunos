# 📚 Sistema de Gerenciamento de Alunos

Um projeto simples em Python para gerenciamento de alunos, permitindo visualizar, adicionar e verificar o status (aprovado ou reprovado) com base nas notas.

---

## 🚀 Funcionalidades

* 📋 Listar alunos cadastrados
* ➕ Adicionar novos alunos
* ✅ Verificar alunos aprovados e reprovados
* ❌ Encerrar o sistema

---

## 🛠️ Tecnologias utilizadas

* Python 3
* Estruturas de dados (listas e dicionários)
* Estruturas de repetição (`for`, `while`)
* Tratamento de erros com `try/except`

---

## ▶️ Como executar o projeto

1. Clone este repositório:

```bash
git clone https://github.com/miguelkk1/gerenciamento-de-alunos
```

2. Acesse a pasta do projeto:

```bash
cd sua-pasta
```

3. Execute o arquivo:

```bash
python nome_do_arquivo.py
```

---

## 💡 Lógica do projeto

O sistema utiliza uma lista de dicionários para armazenar os dados dos alunos, onde cada aluno possui:

* Nome
* Nota

Com base na nota:

* Nota ≥ 7 → Aprovado
* Nota < 7 → Reprovado
